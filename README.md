# Let make HU Student rants and share there spots

## Smartlook phase-1 integration

Set the Smartlook project key before running a native build:

```bash
EXPO_PUBLIC_SMARTLOOK_PROJECT_KEY=your-smartlook-project-key
```

Notes:

- Smartlook works in Expo development builds and release builds.
- Smartlook does not run inside Expo Go.
- After installing native modules, rebuild the app (`expo run:android` / `expo run:ios`).
